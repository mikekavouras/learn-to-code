Variable

Wikipedia: A storage location and an associated symbolic name (an identifier) which contains some known or unknown quantity or information, a value.

The most basic definition of a variable is: A name for something. More specifically, a way to identify something (a value) by a specific name.

For instance, let's think about a person's name. A person's name is a word used to reference an entire person. The word itself doesn't tell us anything specific about the person, but it gives us a way to reference one specific person in a crowd full of people.

Let's create a new variable called personsName. 


var personsName;

Variables aren't really useful until they have a value. For instance, the name "Sam" doesn't mean very much unless there is a person associated with it.

Let's create a new variable without a value. 


var sam;

Now, let's add a value to our new variable. 


sam = new Person();

You can actually combine the 2 lines above. The following example creates a new variable, sam, and gives it a value, all at the same time. 


var sam = new Person();

At this point, the variable sam is referencing a person, and we can use this variable to access information about that specific person. 


sam.name; // "Sam"
sam.age; // 17
sam.height; // 68

We use the var keyword to create a new variable. Once it's been created, you can drop the var. 


var one = 1;
one = 2;
one = 3;

one; // 3

A variable's name is unique. If you attempt to use the same variable more than once, each new declaration will override the previous one. 


sam = "dog";
sam = 1;

sam; // 1

sam is no longer referencing a person, but instead the number 1. Let's make sam a person again. 


sam = new Person();
